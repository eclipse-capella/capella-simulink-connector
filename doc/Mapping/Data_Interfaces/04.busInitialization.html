Bus creation

We would want to create a bus for the following case:
  1. Class with multiple data
  2. Exchange item grouping multiple Class
  3. Functional exchange with multiple Class or Exchange Item
  4. Category to group functional exchange
Information from IFE Example - [CDB] Play Video Movie - Logical

Another approach to handle "generalization" is to create a bus element of data type "Media". This approach leads to a more concise way to create the structureand if Media evolves it's automatically impacted in all structures that are herited from "Media".
