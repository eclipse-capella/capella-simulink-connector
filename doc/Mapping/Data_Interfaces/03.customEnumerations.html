Custom Enumeration

Here we create custom enumerations.
In Simulink we can define an equivalent enumeration object.
Link to Documentation
SHOULD WE PREFIX THE ENUMERATION NAME ?
STREAM_CONTROL = Simulink.data.dictionary.EnumTypeDefinition;
removeEnumeral(STREAM_CONTROL,1);
appendEnumeral(STREAM_CONTROL,'PAUSE',0,'');
appendEnumeral(STREAM_CONTROL,'STOP',1,'');
appendEnumeral(STREAM_CONTROL,'PLAY',2,'');
% With no other information we affect the value in the order of the list
CLASS = Simulink.data.dictionary.EnumTypeDefinition;
removeEnumeral(CLASS,1);
appendEnumeral(CLASS,'BUSINESS',0,'');
appendEnumeral(CLASS,'PREMIUM',1,'');
appendEnumeral(CLASS,'ECONOMY',2,'');
% With no other information we affect the value in the order of the list
