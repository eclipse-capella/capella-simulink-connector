Custom physical quantities

Here we create custom physical quantities.
The physical quantity "Duration" is Integer and its unit is specified as "s" (second).
Link to Documentation
In Simulink we can define an equivalent physical quantity using Simulink.Signal. The limitation is that it only applies to signal (carried by lines).
Duration = Simulink.Signal;
Duration.Unit = 's';
Duration.DataType = 'Int';
Duration
WARNING !!!
The units systems in Simulink are the International Systems
If the unit name in Capella does not match exactly unit expression in Simulink we will get an error later on.
