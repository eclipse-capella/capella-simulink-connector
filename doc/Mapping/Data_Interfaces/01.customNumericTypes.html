Custom numeric type

Here we create custom numeric type.
The class "Command" has one property of type "ID". This "ID" type is defined as an INTEGER Numeric Type.
Link to Documentation
In Simulink we can define an equivalent numeric type.
ID = Simulink.NumericType;
ID.DataTypeMode = "Fixed-point: slope and bias scaling";
ID.Bias = 0;
ID.Signedness = "Signed";
ID.WordLength = 64;
ID.IsAlias = true;
ID
The same applies for
Int = Simulink.NumericType;
Int.DataTypeMode = "Fixed-point: slope and bias scaling";
Int.Bias = 0;
Int.Signedness = "Signed";
Int.WordLength = 64;
Int.IsAlias = true;
Int
