Allocate types and parameters to data dictionnary

Once all the data types and parameters are created in the base workspace, the goal is to store them in adequate data dictionnary (DD).
The first approach could be to create one DD to store all the data.
The second is to create one DD per Data Package.
The example below illustrates the second choice.
varList = {'Language', 'Int', 'ID','Duration'};
createDDAndImportData('Common', varList)
varList = {'busClass_Media', ...'busClass_Audio_File',...
...'busClass_AudioVideo_File',
'busClass_Frame',...
'busClass_Movie', 'busClass_Stream',...
'STREAM_CONTROL'};
createDDAndImportData('Media', varList)
varList = {'busClass_Control_Command', 'busClass_Interactor',...
'busClass_Command','busClass_VOD_Control_Command'};
createDDAndImportData('Commands', varList)
varList = {'busClass_Seat'};
createDDAndImportData('Business_Entities', varList)
function createDDAndImportData(DDName, varList)
mySLDD = Simulink.data.dictionary.create([DDName, '.sldd']);
importFromBaseWorkspace(mySLDD,'varList',varList);
mySLDD.saveChanges();
mySLDD.close()
end
